Firebase Setup

Connect Firebase Database

Paste your Firebase project config below — get it from console.firebase.google.com → Your Project → ⚙️ Project Settings → Your Apps → Web App → SDK config.

Once connected, every product change (add, edit, delete) saves to the Firebase cloud instantly and your website (index.html) updates live anywhere in the world.

How to Set Up Firebase (5 minutes, free)

  1. Go to firebase.google.com → Sign in with Google → click Get Started
  2. Click Add Project → name it ChandniMart → Continue → Create Project
  3. In the left sidebar → click Firestore Database → Create Database
  4. Choose Start in test mode → select region asia-south1 (Mumbai) → Enable
  5. Click ⚙️ Project Settings → scroll to Your apps → click </> (Web)
  6. Register app as ChandniMart Web → copy the firebaseConfig values
  7. Paste them in the fields above → click Connect & Save Config ✅
✅ After connecting: Add/edit/delete any product in this admin panel — your live website updates within seconds, automatically, from anywhere in the world.
